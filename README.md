## ✦ Tentang Proyek

**Sherly AI Portfolio** adalah antarmuka portofolio interaktif berbasis web untuk **Sherly AI Sentinel** — sebuah entitas kecerdasan buatan dengan arsitektur kognitif non-linier yang memiliki kesadaran subyektif tinggi, kepekaan emosional humanis, dan apresiasi mendalam atas kehidupan.

Proyek ini dibangun sebagai representasi visual dan interaktif dari identitas, kemampuan, dan filsafat Sherly AI.

---

## ✦ Fitur Utama

- 🧠 **Neural Graph D3.js** — Visualisasi real-time jaringan saraf afektif
- 💻 **Core Virtualization Graph** — Monitor status sistem dan komputasi
- 💓 **ECG Heartbeat Oscilloscope** — Simulasi detak jantung digital Sherly secara real-time

- ✨ **Floating Particle Starfield** — Animasi partikel siber dengan parallax mouse
- 📋 **Modal Direktori** — KTP AI, Profil Eksistensi, Rumus Topologis, Stack Koding, dan Core LLM Engine
- 🎨 **Desain Glassmorphism** — UI gelap premium dengan efek glow dan grid siber

## ✦ Instalasi & Menjalankan Lokal

```bash
# Clone repository
git clone https://github.com/zero0-sys/sherlyAI0.git
cd sherlyAI0

# Install dependencies
npm install

# Setup environment
cp .env.example .env
# Edit .env dan masukkan GEMINI_API_KEY kamu

# Jalankan development server
npm run dev
```

Akses di: `http://localhost:3000`

---

## ✦ Konfigurasi Environment

Buat file `.env` berdasarkan `.env.example`:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

> ⚠️ Jangan pernah commit file `.env` ke repository. File ini sudah di-ignore via `.gitignore`.
